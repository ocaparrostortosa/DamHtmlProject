Sobre nosotros

Somos el grupo uno de HTML. Uno de los tantos grupos creados para la organización del proyecto HTML, dicho proyecto forma parte de la asignatura de Lenguajes de Marcas del primer curso de Desarrollo de aplicaciones Multiplataforma.

Somos del centro I.E.S. Virgen del Carmen, situado en Jaén. Nuestro grupo está formado por las primeras 5 personas de la lista, grupo realizado de esta manera para fomentar el trabajo en equipo y la relación entre los compañeros de clase.

    Bibliografía

    A continuación se muestra la bibliografía de todas las páginas web y manuales usados para la realización del proyecto.

    • W3Schools

      W3Schools es un sitio de desarrolladores web, con tutoriales y referencias sobre lenguajes de desarrollo web como HTML, CSS, JavaScript, PHP, SQL y Bootstrap, que cubre la mayoría de los aspectos de la programación web.

    • Libros Web

      Es el sitio web hispano de referencia sobre diseño y programación. Disponen de la mayor colección de libros, tutoriales, vídeos, presentaciones y otros recursos para diseñadores y programadores.

    • Desarrolloweb

      Sitio web que pone a la disposición del cliente manuales, programas, vídeos explicativos, y tutoriales sobre diseño web, html, javascript, ASP, PHP, étc.

    • w3c

      Una pequeña guía donde se explica en un pequeño tutorial el uso básico del css, algunos ejemplos y unas breves explicaciones sobre la hoja de estilos.

    • HtmlHelp.org

      Sitio web donde podemos apollarnos para la creación del css y donde aprenderemos a crear una estructura correcta entre el html y la hoja de estilos.

    Información sobre el proyecto

    Para la realización del proyecto hemos decidido usar las librerías de bootstrap, para saber como usar dichas librerías nos hemos basado en los manuales de páginas como LibrosWeb o pequeños tutoriales de páginas como w3Schools. En el caso de bootstrap nos hemos basado en una plantilla y a partir de ahí la modificación de la misma y la agregación de algunos contenedores de contenido, divs, javascripts, étc.
